## Scope Note

These app docs describe the bot that is actually wired in `bot/bot.py`.

Top-level repository docs in `../docs/` are planning and BMAD artifacts. They are useful for roadmap and architecture intent, but they may describe features that are not yet active in the running bot, especially around background scheduling and automatic notifications.
